# 비모수 통계 분석 결과 해석 가이드

> **목적**: 통계 배경지식이 없는 분들도 비모수 분석 결과를 이해하고 의사결정에 활용할 수 있도록 돕는 가이드
> **대상**: 실무 담당자, 의사결정자, 데이터 분석 초보자
> **최종 수정**: 2026-02-13

---

## 📌 이 가이드 사용법

1. **분석 결과를 받았다면**: 해당 분석 방법 섹션으로 바로 이동
2. **p-value가 무엇인지 모른다면**: "기본 개념" 섹션부터 읽기
3. **차트를 이해하고 싶다면**: 각 분석의 "📊 차트 읽는 법" 참조
4. **의사결정이 막막하다면**: "실무 의사결정 예시" 섹션 참조

## 🎓 기본 개념 이해하기

### 1. p-value란? (가장 중요!)

**p-value는 "우연히 이런 결과가 나올 확률"입니다.**

#### 쉬운 이해

- **p-value가 작다 (< 0.05)**: "우연이 아니다. 진짜 차이가 있다!"
- **p-value가 크다 (≥ 0.05)**: "우연일 수도 있다. 확실한 차이라고 보기 어렵다."

#### 기준표

| p-value           | 의미           | 결론                         |
| ----------------- | -------------- | ---------------------------- |
| **< 0.001** | 매우 강한 증거 | ⭐⭐⭐ 확실한 차이 있음      |
| **< 0.01**  | 강한 증거      | ⭐⭐ 차이 있음               |
| **< 0.05**  | 유의한 증거    | ⭐ 차이 있다고 판단          |
| **≥ 0.05** | 증거 부족      | ❌ 차이 있다고 말하기 어려움 |

#### ⚠️ 주의사항

- **p-value만으로는 부족합니다!** 효과 크기도 함께 봐야 합니다.
- p = 0.049와 p = 0.051은 본질적으로 큰 차이가 없습니다.
- "통계적으로 유의하다" ≠ "실질적으로 중요하다"

### 2. 효과 크기 (Effect Size)

**"차이가 얼마나 큰가?"를 나타냅니다.**

| 효과 크기 (r)       | 해석        | 예시                                         |
| ------------------- | ----------- | -------------------------------------------- |
| **0.5 이상**  | 큰 효과     | 신제품이 기존 제품보다 월등히 좋음           |
| **0.3 ~ 0.5** | 중간 효과   | 의미 있는 개선이 있음                        |
| **0.1 ~ 0.3** | 작은 효과   | 미세한 차이, 실무적으로 큰 의미 없을 수 있음 |
| **< 0.1**     | 미미한 효과 | 통계적으로는 유의해도 실질적으로 무의미      |

### 3. 신뢰구간 (Confidence Interval)

**"진짜 값이 이 범위 안에 있을 것이다"라는 구간입니다.**

예시: `평균 매출 증가: 15% [95% CI: 10% ~ 20%]`

- 해석: "95% 확률로 진짜 증가율은 10%에서 20% 사이다"
- 신뢰구간이 0을 포함하지 않으면 → 유의한 차이

## 📊 분석 방법별 결과 해석

### 1. 정규성 검정 (Normality Test)

#### 🎯 무엇을 분석하나요?

데이터가 "종 모양(정규분포)"인지 확인합니다.

#### 📈 결과 읽는 법

```
Shapiro-Wilk Test
- Statistic (W): 0.9845
- p-value: 0.0234
- Is Normal: False
```

**해석**:

- **Is Normal = True** → 데이터가 정규분포입니다 (모수 검정 가능)
- **Is Normal = False** → 데이터가 치우쳐 있습니다 (**비모수 검정 사용**)

#### 📊 차트 읽는 법

**1) 히스토그램 + KDE**

```
   빈도
    │     ╱‾‾╲
    │    ╱    ╲     ← KDE (부드러운 곡선)
    │   ▂▃▅▇▅▃▂    ← 히스토그램
    └──────────── 값
```

- **종 모양**이면 정규분포
- **한쪽으로 치우침**이면 비정규분포

**2) Q-Q Plot (분위수-분위수 그림)**

```
관측값
    │     •
    │   • •
    │  •  •  ← 점들이 빨간 선을 따라가면 정규분포
    │ •    •
    │•      •
    └────────── 이론값
```

- **점들이 직선 위**에 있으면 정규분포
- **직선에서 벗어남**이면 비정규분포

**3) Box Plot (상자그림)**

```
    │
    ├──┬──┐  ← Q3 (75%)
    │  │  │
    │──●──│  ← Median (50%)
    │  │  │
    └──┴──┘  ← Q1 (25%)
       │
```

- **상자가 대칭**이면 정규분포 가능성 높음
- **한쪽으로 치우침**이면 비정규분포

#### 💼 실무 의사결정

- **정규분포 → t-검정, ANOVA 같은 모수 검정 사용 가능**
- **비정규분포 → Mann-Whitney, Kruskal-Wallis 같은 비모수 검정 사용**

### 2. 런 검정 (Runs Test - 무작위성)

#### 🎯 무엇을 분석하나요?

데이터가 무작위로 분포하는지, 아니면 어떤 패턴이 있는지 확인합니다.

#### 📈 결과 읽는 법

```
Runs Test
- Runs: 8
- Expected: 60.0
- Z: -9.62
- p-value: 0.0000
```

**해석**:

- **p < 0.05** → 무작위가 **아닙니다**. 어떤 패턴이 존재합니다.
- **p ≥ 0.05** → 무작위입니다. 패턴이 없습니다.

#### 📊 차트 읽는 법

**원본 데이터 + 중앙값 기준선**

```
값
 │  • ••    • •• ••
 │ ────────────────  ← 중앙값 (빨간 점선)
 │••    ••      •
 └──────────────── 시간
```

- **위아래로 자주 바뀜** → 무작위
- **한쪽에 몰림** → 패턴 있음 (추세, 주기성 등)

**이진 변환 (Above/Below)**

```
Above │ ▬▬▬    ▬▬▬ ▬▬
      │
Below │    ▬▬▬      ▬
      └──────────────
```

- **Run 개수가 적음** → 구간이 뭉쳐 있음 (패턴 존재)
- **Run 개수가 많음** → 자주 변함 (무작위)

#### 💼 실무 의사결정

- **패턴 발견** → 시계열 분석, 추세 분석 필요
- **무작위** → 독립적인 데이터, 일반적인 통계 분석 가능

### 3. 추세 분석 (Mann-Kendall Trend)

#### 🎯 무엇을 분석하나요?

시간에 따라 데이터가 **증가하는지, 감소하는지, 변화가 없는지** 확인합니다.

#### 📈 결과 읽는 법

```
Mann-Kendall Test
- Tau: 0.524
- p-value: 0.0001
- Slope: 0.0523
- Trend: increasing
```

**해석**:

- **Trend: increasing** → 증가 추세
- **Trend: decreasing** → 감소 추세
- **Trend: no trend** → 추세 없음
- **Slope**: 단위 시간당 변화량 (양수면 증가, 음수면 감소)

#### 📊 차트 읽는 법

```
값
 │        •
 │      • │ •
 │    •   │   •   ← 원본 데이터 (파란색)
 │  •     │     •
 │ ╱      │      ╲
 │╱───────────────╲ ← Sen's Slope (빨간 선)
 └────────────────── 시간
```

- **빨간 선이 위로** → 증가 추세
- **빨간 선이 아래로** → 감소 추세
- **빨간 선이 평평** → 추세 없음

#### 💼 실무 의사결정

- **증가 추세**: 매출 증가, 사용자 증가 → 긍정적
- **감소 추세**: 이탈률 증가, 만족도 감소 → 대책 필요
- **추세 없음**: 현상 유지 → 현재 전략 지속 또는 변화 시도

### 4. 변곡점 탐지 (Pettitt Test)

#### 🎯 무엇을 분석하나요?

데이터의 흐름이 **갑자기 바뀌는 지점**을 찾습니다.

#### 📈 결과 읽는 법

```
Pettitt Test
- Change Point: 45
- Statistic (K): 1234.0
- p-value: 0.0023
- Median Before: 32.5
- Median After: 48.7
```

**해석**:

- **p < 0.05** → 변곡점이 존재합니다
- **Change Point: 45** → 45번째 데이터 지점에서 변화 발생
- **Median Before vs After** → 변화 전후 수준 차이

#### 📊 차트 읽는 법

```
값
 │                 •••
 │               ••───  ← After (주황 선)
 │    •••      •
 │  ••───    •          ← Before (초록 선)
 │••    │
 │      ▼ 변곡점 (빨간 점선)
 └────────────────── 시간
```

- **빨간 점선** = 변곡점 위치
- **초록 선** = 변화 전 중앙값
- **주황 선** = 변화 후 중앙값

#### 💼 실무 의사결정

- **정책 도입 효과 분석**: "9월 1일 신정책 이후 매출이 변했나?"
- **시스템 장애 탐지**: "언제부터 서버 응답 시간이 느려졌나?"
- **마케팅 효과**: "광고 캠페인 후 사용자 행동이 바뀌었나?"

### 5. 다중 구간 분할 (PELT)

#### 🎯 무엇을 분석하나요?

데이터의 패턴이 바뀌는 **여러 지점**을 동시에 찾습니다.

#### 📈 결과 읽는 법

```
PELT Changepoints
- Changepoints: [23, 67, 112]
- Number of Segments: 4
```

**해석**:

- **Changepoints**: 패턴이 바뀌는 지점들
- **Number of Segments**: 전체 구간 개수 (변곡점 + 1)

#### 📊 차트 읽는 법

**상단: 원본 데이터 + 구간별 중앙값**

```
값
 │  ••• ▬▬  ••▬▬  •••▬▬
 │ •      •       •
 │ │      │       │
 │ ▼      ▼       ▼  ← 변곡점들
 └────────────────────
```

- **빨간 가로선** = 각 구간의 중앙값
- **빨간 점선** = 변곡점 위치

**하단: 구간별 분포**

```
  Box Plot
    ┌─┐  ┌──┐ ┌─┐
    │ │  │  │ │ │
    └─┘  └──┘ └─┘
    Seg1 Seg2 Seg3
```

- 각 구간의 분포 차이를 상자그림으로 비교

#### 💼 실무 의사결정

- **여러 정책/이벤트 효과 분석**: "분기별 주요 이벤트가 매출에 미친 영향"
- **계절성 패턴 발견**: "성수기/비수기 구분점 자동 탐지"
- **이상 구간 식별**: "평소와 다른 행동 패턴을 보인 기간"

### 6. 두 독립 그룹 비교 (Mann-Whitney U Test)

#### 🎯 무엇을 분석하나요?

**서로 다른 두 그룹**의 차이를 비교합니다.
예: A/B 테스트, 남성 vs 여성, 서울 vs 지방

#### 📈 결과 읽는 법

```
Mann-Whitney U Test
- Statistic (U): 1234.5
- Z: -4.23
- p-value: 0.0000
- Effect Size (r): 0.42 (중간)
- Cliff's Delta: 0.35
```

**해석**:

- **p < 0.05** → 두 그룹은 **통계적으로 차이가 있습니다**
- **Effect Size (r)**:
  - r ≥ 0.5: 큰 차이
  - r ≥ 0.3: 중간 차이
  - r < 0.3: 작은 차이

#### 📊 차트 읽는 법

**Box Plot + Scatter (좌측)**

```
값
 │   ○○
 │ ┌─┼─┐  ┌─┼─┐
 │ │ ● │  │ ● │  ← 상자: IQR, ●: 중앙값
 │ └─┼─┘  └─┼─┘
 │   ○○    ○○○   ← 점: 개별 데이터
 └─────────────
   Group A  Group B
```

- **상자가 겹치지 않으면** → 차이가 큼
- **중앙값(●) 위치 비교** → 어느 그룹이 높은지
- **점들의 분포** → 데이터 퍼짐 정도

**Rank Distribution (우측)**

```
빈도
 │   ▂▃▅▇▅▃▂
 │ ▃▅▇▅▃▂
 │
 └──────────── 순위
   A     B
```

- **분포가 분리**되면 → 차이가 명확
- **겹침이 많으면** → 차이가 작음

#### 💼 실무 의사결정

- **A/B 테스트**: "신규 UI(A)가 기존 UI(B)보다 클릭률이 높은가?"
  - p < 0.05, r = 0.5 → **신규 UI 채택**
  - p < 0.05, r = 0.1 → 통계적 유의하지만 실질적 효과 작음
- **고객 세그먼트**: "VIP 고객과 일반 고객의 구매액 차이"
  - 차이 있음 → VIP 맞춤 마케팅 강화

### 7. 두 분포 비교 (K-S Test)

#### 🎯 무엇을 분석하나요?

두 그룹의 **분포 모양** 자체가 다른지 확인합니다.

#### 📈 결과 읽는 법

```
K-S Test
- Statistic (D): 0.342
- p-value: 0.0023
```

**해석**:

- **p < 0.05** → 두 그룹의 분포가 다릅니다
- **Statistic (D)**: 0에 가까울수록 비슷한 분포

#### 📊 차트 읽는 법

**ECDF (Empirical Cumulative Distribution Function)**

```
누적확률
1.0 │        ╱‾‾‾‾‾‾  ← Group B (빨강)
    │       ╱
0.5 │   ╱‾‾‾
    │  ╱            ← Group A (파랑)
0.0 │╱
    └──────────────── 값
```

- **두 선이 멀리 떨어짐** → 분포가 많이 다름
- **최대 수직 거리** = D 통계량

#### 💼 실무 의사결정

- Mann-Whitney는 **중앙값 차이**만 봄
- K-S는 **전체 분포 형태**를 봄
- 중앙값은 같아도 분포가 다를 수 있음 → 둘 다 확인 권장

### 8. 짝지어진 그룹 비교 (Wilcoxon Signed Rank)

#### 🎯 무엇을 분석하나요?

**동일한 대상**의 전/후 변화를 비교합니다.
예: 약물 투여 전후, 교육 프로그램 전후

#### 📈 결과 읽는 법

```
Wilcoxon Signed Rank Test
- Statistic (T): 234.5
- Z: -3.12
- p-value: 0.0018
- Effect Size (r): 0.38 (중간)
- Median Difference: 5.3
```

**해석**:

- **p < 0.05** → 전후에 **유의미한 변화**가 있었습니다
- **Median Difference > 0** → 증가
- **Median Difference < 0** → 감소

#### 📊 차트 읽는 법

**Spaghetti Plot (좌측)**

```
값
 │     ╱  ← 증가 (초록)
 │   ╱╱
 │  │╲    ← 감소 (빨강)
 │ │  ╲
 └─────
  Before After
```

- **초록 선** = 증가한 경우
- **빨강 선** = 감소한 경우
- **대부분 위로** → 전반적 증가

**Difference Histogram (우측)**

```
빈도
 │      ▇▇▇
 │    ▃▇   ▇▃
 │  ▂▃       ▃▂
 └──│─────────  ← 0 (변화 없음)
    ▼ 중앙값 (빨간 선)
```

- **중앙값이 0보다 오른쪽** → 증가
- **중앙값이 0보다 왼쪽** → 감소

#### 💼 실무 의사결정

- **약물/치료 효과**: "혈압약 복용 후 혈압이 낮아졌나?"
  - p < 0.05, Median Diff = -15 → **약 효과 있음**
- **교육 효과**: "교육 프로그램 후 성적 향상"
  - p < 0.05, r = 0.6 → **교육 효과 크다, 프로그램 확대**

### 9. 부호 검정 (Sign Test)

#### 🎯 무엇을 분석하나요?

변화의 **방향**만 봅니다. (얼마나 변했는지는 무시)

#### 📈 결과 읽는 법

```
Sign Test
- Positive: 18
- Negative: 7
- p-value: 0.0234
```

**해석**:

- **Positive > Negative** → 대부분 증가
- **Positive < Negative** → 대부분 감소
- **p < 0.05** → 증가/감소 경향이 유의함

#### 📊 차트 읽는 법

```
차이값
 │  ▇▇      ← 양수 (초록)
 │  ▇▇▇
 │  ▇
 │─────────  ← 0
 │ ▃        ← 음수 (빨강)
 └─────────
```

- **초록 막대가 많으면** → 증가한 사람이 많음
- **빨강 막대가 많으면** → 감소한 사람이 많음

#### 💼 실무 의사결정

- Wilcoxon보다 **단순하지만 검정력 낮음**
- 데이터가 매우 치우쳐 있거나 이상치가 많을 때 유용

---

### 10. 세 독립 그룹 비교 (Kruskal-Wallis)

#### 🎯 무엇을 분석하나요?

**3개 이상의 서로 다른 그룹** 중 적어도 하나는 다른지 확인합니다.

#### 📈 결과 읽는 법

```
Kruskal-Wallis Test
- Statistic (H): 23.45
- p-value: 0.0001
- Eta Squared: 0.234
```

**해석**:

- **p < 0.05** → 적어도 한 그룹은 다릅니다
- **하지만 어느 그룹이 다른지는 모릅니다!**
- → 사후 검정(Post-hoc: Dunn Test) 필요

#### 📊 차트 읽는 법

```
값
 │  ┌─┐  ┌──┐ ┌─┐
 │  │●│  │ ●│ │●│
 │  └─┘  └──┘ └─┘
 │  ○○○  ○○○  ○○
 └─────────────────
   G1    G2   G3
```

- **상자 위치 비교** → 어느 그룹이 높은지/낮은지
- **상자 크기** → 그룹 내 편차

#### 💼 실무 의사결정

- **지역별 비교**: "서울/경기/지방의 매출 차이"
  - p < 0.05 → 지역별 차이 있음
  - 사후 검정 → "서울 > 경기, 서울 > 지방, 경기 = 지방"
  - 결론: **서울 집중 마케팅 강화**
- **연령대별 선호도**: "20대/30대/40대의 제품 만족도"

---

### 11. 반복 측정 비교 (Friedman Test)

#### 🎯 무엇을 분석하나요?

**동일한 대상**을 3번 이상 반복 측정했을 때 차이를 봅니다.

#### 📈 결과 읽는 법

```
Friedman Test
- Statistic (χ²): 18.34
- p-value: 0.0003
- Kendall's W: 0.456
```

**해석**:

- **p < 0.05** → 시점 간 차이가 있습니다
- **Kendall's W**: 0에 가까우면 차이 없음, 1에 가까우면 차이 큼

#### 📊 차트 읽는 법

**Spaghetti Plot (좌측)**

```
값
 │    •──•╲╲
 │   ╱    ╲╲╲
 │  •      •╲  ← 개별 대상들
 │ ╱        ╲╲
 │●──────────●  ← 중앙값 (빨강)
 └─────────────
  T1   T2   T3
```

- **빨간 선** = 시점별 중앙값 변화
- **회색 선들** = 개별 대상의 변화 패턴

**Box Plot (우측)**

```
값
 │ ┌─┐ ┌──┐ ┌─┐
 │ │●│ │ ●│ │●│
 │ └─┘ └──┘ └─┘
 └─────────────
  T1   T2   T3
```

#### 💼 실무 의사결정

- **분기별 성과**: "1Q/2Q/3Q/4Q 매출 비교"
  - p < 0.05, W = 0.7 → **계절성 강함**
- **단계별 효과**: "3단계 프로그램 각 단계별 효과"
  - T1 < T2 < T3 → **누적 효과 있음**

---

### 12. 상관 행렬 (Correlation Matrix)

#### 🎯 무엇을 분석하나요?

여러 변수들 간의 **관계**를 한눈에 봅니다.

#### 📈 결과 읽는 법

```
Spearman Correlation Matrix
             feature_1  feature_2  feature_3
feature_1       1.00      0.67      -0.23
feature_2       0.67      1.00       0.12
feature_3      -0.23      0.12       1.00
```

**상관계수 해석**:

- **1.0** = 완벽한 양의 상관 (하나 증가 → 다른 것도 증가)
- **0.7 ~ 1.0** = 강한 양의 상관
- **0.3 ~ 0.7** = 중간 양의 상관
- **0.0 ~ 0.3** = 약한 상관
- **-0.3 ~ 0.0** = 약한 음의 상관
- **-1.0** = 완벽한 음의 상관 (하나 증가 → 다른 것 감소)

#### 📊 차트 읽는 법

**Heatmap (히트맵)**

```
          f1   f2   f3
    f1  [1.0][0.7][-0.2]
    f2  [0.7][1.0][0.1]
    f3  [-0.2][0.1][1.0]

    색상: 빨강 → 노랑 → 초록
         -1.0   0.0   1.0
```

- **초록색 (양수)** = 같이 증가/감소
- **빨강색 (음수)** = 반대로 움직임
- **노란색 (0 근처)** = 관계 없음

**p-value 히트맵**

```
p-value가 작을수록 (초록) → 신뢰할 수 있는 상관
p-value가 클수록 (빨강) → 우연일 수 있음
```

#### 💼 실무 의사결정

- **다중공선성 체크**: 상관 > 0.8이면 변수 중복 가능성
- **주요 영향 변수 발견**: "매출과 가장 상관 높은 변수는?"
- **예측 모델 변수 선택**: 상관 높은 변수 우선 사용

### 13. Kendall 상관 & Distance 상관

#### 🎯 무엇을 분석하나요?

- **Kendall**: 순위 동점이 많을 때 더 정확한 상관
- **Distance**: 곡선 관계(비선형)도 찾아냅니다

#### 📈 결과 읽는 법

```
Kendall Tau: 0.5763
Distance Corr: 0.7187
```

**해석**:

- **Kendall Tau**: -1 ~ 1 (Spearman과 유사, 보수적)
- **Distance Corr**: 0 ~ 1
  - **0** = 완전 독립
  - **1** = 완전 종속
  - 비선형 관계도 탐지!

#### 📊 차트 읽는 법

**Scatter Plot**

```
Y
 │    •  •
 │  •  •  •
 │ •      •  ← 곡선 패턴
 │•        •
 └──────────── X
```

- **직선 패턴** → Spearman/Kendall이 잘 잡음
- **곡선 패턴** → Distance Correlation이 더 잘 잡음

#### 💼 실무 의사결정

- **Spearman = 0.1, Distance = 0.7** → 비선형 관계 존재!
- 예: "광고비와 매출이 일정 수준까지만 증가" (포화 효과)

### 14. Bootstrap 신뢰구간

#### 🎯 무엇을 분석하나요?

데이터가 적을 때, 통계량(평균, 중앙값 등)의 **신뢰구간**을 추정합니다.

#### 📈 결과 읽는 법

```
Bootstrap 95% CI
- Observed: 45.3
- CI Lower: 42.1
- CI Upper: 48.7
```

**해석**:

- **관측값: 45.3** = 실제 데이터에서 계산한 값
- **95% CI: [42.1, 48.7]** = "95% 확률로 진짜 값은 이 범위 안"

#### 📊 차트 읽는 법

```
빈도
 │      ▇▇▇
 │    ▃▇   ▇▃
 │  ▂▃       ▃▂  ← Bootstrap 분포
 │ │         │
 │ ▼         ▼
 └─────────────
  42.1  45.3  48.7
   ▼     ▼     ▼
  Lower Obs  Upper
```

- **빨간 선** = 관측값
- **주황 음영** = 95% 신뢰구간
- 구간이 좁을수록 → 정확한 추정

#### 💼 실무 의사결정

- **표본이 작을 때 유용**: "30명 고객만으로 전체 만족도 추정"
- **CI가 0을 포함 안 하면** → 유의한 차이

### 15. Permutation 검정 (순열 검정)

#### 🎯 무엇을 분석하나요?

정규분포 가정 없이 **두 그룹의 차이가 우연인지** 검정합니다.

#### 📈 결과 읽는 법

```
Permutation Test
- Observed Difference: 12.3
- p-value: 0.0021
```

**해석**:

- **p < 0.05** → 차이가 우연이 아닙니다
- **Observed Difference** = 실제 관측된 차이

#### 📊 차트 읽는 법

```
빈도
 │    ▃▇▇▇▃
 │  ▂▃     ▃▂  ← 무작위로 섞은 결과들
 │ ▂           ▂
 └──────│──────
        ▼
   관측값 (빨간 선)
```

- **빨간 선이 분포 밖** → 우연이 아님 (p 작음)
- **빨간 선이 분포 안** → 우연일 수 있음 (p 큼)

#### 💼 실무 의사결정

- **소표본 A/B 테스트**: 정규분포 가정 없이 검정 가능
- **복잡한 통계량**: 평균 차이뿐 아니라 어떤 통계량도 검정 가능

### 시나리오 1: 신제품 출시 효과 분석

**상황**: 신제품 출시 전후 3개월 매출 데이터

**분석 순서**:

1. **Pettitt Test**: 변곡점 탐지

   - 결과: 변곡점 = 출시일, p = 0.001
   - **해석**: 출시일을 기점으로 매출 패턴 변화
2. **Wilcoxon Paired Test**: 전후 비교

   - 결과: p = 0.003, Median Diff = +23%, r = 0.5
   - **해석**: 출시 후 매출 23% 증가, 효과 큼
3. **Mann-Kendall Test**: 출시 후 추세

   - 결과: Trend = increasing, p = 0.01
   - **해석**: 출시 후에도 지속적 증가 추세

**의사결정**: ✅ **신제품 성공! 생산량 증대 및 마케팅 강화**

### 시나리오 2: 지역별 마케팅 전략

**상황**: 서울/경기/지방의 고객 만족도 차이

**분석 순서**:

1. **정규성 검정**

   - 결과: Is Normal = False
   - **해석**: 비모수 검정 사용
2. **Kruskal-Wallis Test**

   - 결과: p < 0.001
   - **해석**: 지역 간 차이 있음
3. **Dunn Post-hoc**

   - 서울 vs 경기: p = 0.001
   - 서울 vs 지방: p = 0.0001
   - 경기 vs 지방: p = 0.523
   - **해석**: 서울만 유의하게 높음

**의사결정**: ✅ **서울 중심 프리미엄 전략, 경기/지방은 가격 경쟁력 강화**

---

### 시나리오 3: A/B 테스트 결과 판단

**상황**: 새 UI(A) vs 기존 UI(B) 클릭률 비교

**분석 결과**:

```
Mann-Whitney U Test
- p-value: 0.042 (< 0.05) ✓
- Effect Size (r): 0.12 (작음)
- Cliff's Delta: 0.08
- 평균 차이: 2.3%p (5.1% → 7.4%)
```

**판단**:

- **통계적 유의성**: 있음 (p < 0.05)
- **효과 크기**: 작음 (r = 0.12)
- **실질적 차이**: 2.3%p

**의사결정 고려사항**:

- ✅ **채택**: UI 변경 비용이 낮고, 2.3%p 향상이 큰 매출 증가로 이어진다면
- ❌ **기각**: UI 변경 비용이 크고, 2.3%p 차이가 실질적으로 미미하다면

**결론**: ⚖️ **비용-편익 분석 후 결정 (통계만으로는 부족)**

## ❓ 자주 묻는 질문 (FAQ)

### Q1. p-value가 0.051인데, 0.05보다 살짝 커서 무의미한가요?

**A**: 아닙니다!

- p = 0.049와 p = 0.051은 본질적으로 큰 차이가 없습니다.
- 0.05는 **임의의 기준선**일 뿐, 절대적 경계가 아닙니다.
- **효과 크기, 신뢰구간, 실질적 중요도**를 함께 고려하세요.

### Q2. 통계적으로 유의한데 효과 크기가 작으면?

**A**: 표본이 매우 크면 작은 차이도 유의하게 나옵니다.

- **예**: 1만 명 데이터에서 0.1% 차이도 p < 0.05 가능
- **실무 판단**: 효과 크기(r, Cliff's δ)가 작으면 실질적 의미 없을 수 있음
- **결론**: **통계적 유의성 ≠ 실질적 중요성**

### Q3. 정규분포인지 아닌지 판단이 애매할 때는?

**A**: 애매하면 **비모수 검정 사용을 권장**합니다.

- 비모수 검정은 정규분포에서도 잘 작동 (검정력 손실 ~5%)
- 비정규분포에서 모수 검정은 오류 가능성 높음
- **안전한 선택 = 비모수 검정**

### Q4. 상관관계가 있다고 인과관계도 있는 건가요?

**A**: 절대 아닙니다!

- **상관**: "같이 움직인다"
- **인과**: "하나가 다른 하나를 일으킨다"
- **예**: 아이스크림 판매량 ↔ 익사 사고 (상관 O, 인과 X, 진짜 원인은 '여름')
- **인과 확인**: 실험, 시간 순서, 이론적 근거 필요

### Q5. 어떤 그래프를 봐야 하나요?

**A**: 분석 목적에 따라 다릅니다.

- **분포 확인**: 히스토그램, Box Plot, Violin Plot
- **비교**: Box Plot (그룹 간), Spaghetti Plot (전후)
- **추세**: 시계열 라인 차트 + Sen's Slope
- **상관**: Scatter Plot, Heatmap
- **분포 차이**: ECDF (K-S Test)

### Q6. 표본 크기가 얼마나 필요한가요?

**A**: 분석 방법에 따라 다릅니다.

- **최소 권장**:
  - Mann-Whitney: 각 그룹 20개 이상
  - Wilcoxon: 15쌍 이상
  - Kruskal-Wallis: 각 그룹 5개 이상
- **표본이 작으면**: Bootstrap, Permutation Test 활용
- **표본이 매우 크면**: 작은 차이도 유의하게 나옴 (효과 크기 필수 확인)

### Q7. 여러 번 검정하면 문제가 되나요?

**A**: 네, **다중 검정 문제**가 발생합니다.

- 20번 검정하면 1번은 우연히 p < 0.05 나옴
- **해결책**: Bonferroni 보정, FDR 보정
  - 예: 10번 검정 → 유의 수준 0.05 / 10 = 0.005
- **권장**: 사전에 가설을 명확히 정하고, 필요한 검정만 수행

### Q8. 비모수 검정이 항상 더 안전한가요?

**A**: 대부분 그렇지만, 단점도 있습니다.

- **장점**:
  - 분포 가정 불필요
  - 이상치에 강건
  - 순서형 데이터에도 적용 가능
- **단점**:
  - 검정력이 약간 낮음 (~5%)
  - 평균이 아닌 중앙값 비교 (극단값 영향 무시)
- **결론**: 대부분 경우 비모수 검정이 안전하고 실용적

### Q9. 차트의 색깔이 의미가 있나요?

**A**: 네, 일반적인 컨벤션이 있습니다.

- **초록색**: 증가, 긍정, 높음
- **빨간색**: 감소, 부정, 낮음
- **파란색**: 원본 데이터, 그룹 A
- **주황색**: 추정값, 그룹 B
- **회색**: 개별 데이터 포인트
- 하지만 **범례를 항상 확인**하세요!

### Q10. 결과를 보고서로 작성할 때 주의사항은?

**A**: 다음 원칙을 따르세요.

**✅ 해야 할 것**:

- p-value + 효과 크기 + 신뢰구간 모두 보고
- 차트 포함 (시각적 이해 향상)
- 비전문가도 이해할 수 있는 언어 사용
- 제한사항 명시 (표본 크기, 가정 등)

**❌ 하지 말아야 할 것**:

- p-value만 보고하기
- "유의하다/유의하지 않다"만 쓰기
- 효과 크기 무시하기
- 인과관계 주장 (상관관계만 있을 때)
